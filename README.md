# 💳 Loan Default Risk Prediction System

An AI-powered web application that predicts loan default risk using Logistic Regression and provides decision support recommendations.

---

## 🚀 Features

- Predicts loan default probability
- Categorizes risk (Low / Moderate / High)
- Dynamic Risk Meter visualization
- Loan approval recommendation
- Suggested safe loan amount
- User-friendly web interface
- REST API backend

---

## 🧠 Machine Learning Model

- Algorithm: Logistic Regression
- Dataset: Credit Card Default Dataset (30,000 records)
- Preprocessing:
  - Feature Scaling (StandardScaler)
  - Feature Selection (SelectKBest)
- Evaluation:
  - Accuracy ~80%
  - ROC-AUC score calculated

---

## 🏗 Architecture

Frontend (HTML + Tailwind CSS)
↓
Flask Backend API
↓
Preprocessing Pipeline
↓
Logistic Regression Model
↓
Risk Prediction + Decision Logic

---

## 🛠 Tech Stack

- Flask
- Scikit-learn

- NumPy
- Tailwind CSS
